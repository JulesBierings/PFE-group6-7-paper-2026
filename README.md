# Affordability of Houses in the Netherlands

## Project Description
This project analyses whether housing prices in the Netherlands have increased faster than employee incomes between 2015 and 2024. Housing affordability is measured using the ratio between average WOZ values and average employee income.

## How to Reproduce the Project
1. Clone or download this repository.
2. Open the project in RStudio.
3. Install the required packages if necessary.
4. Run run-all.R or Knit the Final-Report.Rmd file.
5. The script downloads/loads the data, cleans the datasets, creates the figures and tables, and generates the final report.
